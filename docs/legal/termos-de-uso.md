## 2. Definições

- **Plataforma/Adelina:** o sistema de gestão hoteleira (PMS), incluindo gestão de reservas,
  calendário, channel manager (Airbnb/Booking via iCal), hóspedes, cobranças e módulos
  financeiros, acessível via navegador.
- **Conta:** o cadastro do Assinante, vinculado a um e-mail e a um estabelecimento (pousada/hotel).
- **Assinatura:** o plano contratado de uso recorrente da Plataforma.
- **Hóspede:** o cliente final do Assinante, cujos dados são tratados pelo Assinante na Plataforma.
- **Conteúdo do Cliente:** todos os dados inseridos pelo Assinante (reservas, hóspedes, valores etc.).
